Overview of C++

C++ is a widely-used programming language known for its performance and versatility. It is an extension of the C programming language and is often used for:

  • System/software development
  • Game development
  • Embedded systems

Basic Syntax

Structure of a C++ Program

A C++ program consists of:

  • Preprocessor directives
  • Functions

The main function is the entry point of the program where execution begins.

#include <iostream>
int main(){
  // Main code goes here return 0;
}

Comments

Comments are used for documentation and explanation within the code. They can be:

  • Single-line: // Comment
  • Multi-line: /* Comment */

Data Types and Variables

Basic Data Types

C++ supports various data types such as:

  • Integers
  • Floating-point numbers
  • Characters
  • Booleans
int age = 25;
float height = 5.9;
char grade = 'A';
bool isStudent = true;

Declaring Variables

Variables are used to store data. They must be declared with a specific data type before they can be used.

Operators

Arithmetic Operators

C++ supports arithmetic operations like:

  • Addition
  • Subtraction
  • Multiplication
  • Division
  • Modulus
int result = 10 + 5; // Addition

Comparison Operators

Comparison operators are used to compare values, such as:

  • Equal to
  • Not equal to
  • Greater than
  • Less than
bool isEqual = (10 == 5); // Equal to

Logical Operators

Logical operators are used for logical operations, including:

  • Logical AND
  • Logical OR
  • Logical NOT
bool isTrue = (true && false); // Logical AND

Control Flow

Conditional Statements

Conditional statements allow executing code based on conditions. They include:

  • If statements
  • Else statements
  • Else-if statements
int number = 10;
if (number > 0) {
  // Code block if condition is true
}
else {
  // Code block if condition is false
}

Loops

Loops are used to execute a block of code repeatedly. They include:

  • For loops
  • While loops
  • Do-while loops
for (int i = 0; i < 5; i++) {
  // Code block to be repeated
}

Functions

Defining and Calling Functions

Functions are blocks of code that perform a specific task. They are defined using a return type, a function name, and optional parameters. Functions are called by their name followed by parentheses.

#include <iostream>
void sayHello() {
  std::cout << "Hello, World!" << std::endl;
}
int main() {
  sayHello(); // Calling the function
  return 0;
}

Object-Oriented Programming Basics

Classes and Objects

Classes are blueprints for creating objects. They contain data members (attributes) and member functions (methods) to operate on those data members.

#include <iostream>
class Dog {
  public:
  void bark() {
    std::cout << "Woof!" << std::endl;
  }
};
int main() {
  Dog myDog;
  myDog.bark(); // Calling the method
  return 0;
}

Constructors and Destructors

Constructors are special member functions called when an object is created. They initialize the object's data members. Destructors are called when an object is destroyed and are used to clean up resources.

#include <iostream>
class Dog {
  public:
  Dog() { // Constructor
    std::cout << "Dog created" << std::endl;
  }
  ~Dog() { // Destructor
    std::cout << "Dog destroyed" << std::endl;
  }
};
int main() {
  Dog myDog;
  return 0;
}

Standard Library and Common Functions

C++ provides a standard library with common functions and classes for various tasks such as:

  • Input/output
  • String manipulation
  • Data structures

Compilation and Execution

C++ programs are compiled into machine code using a compiler. The compiled code is then executed to produce the desired output.

g++ program.cpp -o program
./program
